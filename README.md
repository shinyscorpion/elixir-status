# Status

Status information for a running system.

## Installation

The package can be installed by adding `status`
to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:status, "~> 0.1"}
  ]
end
```

The docs can
be found at [https://hexdocs.pm/status](https://hexdocs.pm/status).
